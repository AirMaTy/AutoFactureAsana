Documentation Utilisateur – AutoFacture Asana v1.0

Cette application permet de synchroniser automatiquement les factures payées (emails Stripe/Axonaut) avec Asana.

Fonctionnalités

  • Lecture des e-mails non lus dans INBOX/02-STRIPE
  • Extraction des factures avec numéro FXXXXXXXX-XXXX
  • Mise à jour du champ personnalisé “Paiement CG FRANCE” dans les tâches Asana
  • Interface graphique avec bouton unique : Scanner les e-mails
  • Affichage des actions dans une zone de logs (couleur verte = OK, rouge = erreur)

Utilisation

  1. Lancer l’application
  2. Cliquer sur “Scanner les e-mails”
  3. Les mises à jour s’effectuent automatiquement

Comportement

  • Uniquement les mails non lus (UNSEEN) sont traités
  • Un mail peut contenir une facture qui correspond à une tâche unique ou une tâche contenant plusieurs factures
  • Pas de suppression ni déplacement de mail
  • Tout est automatique
