User Documentation – AutoFacture Asana v1.0

This tool automatically updates invoices from email to Asana task field.

Features

  • Scan unread emails in INBOX/02-STRIPE
  • Extract invoice number and client
  • Update Asana field “Paiement CG FRANCE” if task matches
  • Logs displayed in interface

Behavior

  • Only unread emails (UNSEEN) are processed
  • An email may contain an invoice linked to a single task or to a task containing multiple invoices
  • No email is deleted or moved
  • Everything is automatic

How to use

  1. Run the application
  2. Click “Scan Emails”
  3. Let it update invoices automatically
